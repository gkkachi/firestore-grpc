rustls-native-certs allows rustls to use the platform’s native certificate store when operating as a TLS client.

It provides the following functions:

  • A higher level function load_native_certs which returns a rustls::RootCertStore pre-filled from the native certificate store. It is only available if the rustls feature is enabled.
  • A lower level function build_native_certs that lets callers pass their own certificate parsing logic. It is available to all users.
